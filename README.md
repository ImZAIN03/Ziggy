## 🎮 Gameplay

Ziggy is a one-touch reflex game set in a deep-space neon world.

- A **glowing ball** bounces horizontally across the screen, leaving a luminous trail
- Neon **walls with gaps** scroll down from the top — each a different vibrant color
- **Tap anywhere** on the screen to instantly flip the ball's direction
- Thread the ball through each gap to score a point
- **Miss a gap and it's game over** — a particle burst marks your demise
- The game steadily **speeds up** over time, and the gaps get narrower — survive as long as you can

Your current score and all-time best are shown on the game-over screen.

## ✨ Features

| Feature | Details |
|---|---|
| 🌌 **Neon atmosphere** | Deep-space background with scrolling grid lines and twinkling floating particles |
| 💫 **Ball effects** | Multi-layer glow, radial gradient core, speed lines, and a glowing trail |
| 🎨 **Colorful walls** | Walls cycle through cyan, pink, purple, green, and yellow with atmospheric bloom |
| 📳 **Screen shake** | Camera shake on every tap, with a big burst-and-shake on death |
| 🔤 **Orbitron font** | Retro-futuristic UI rendered entirely in the Orbitron typeface |
| 📈 **Progressive difficulty** | Speed scales up to 3.5× and gaps tighten as you survive longer |
| 🏆 **Best score tracking** | Your high score is remembered across the session |

## 🚀 How to Run

### Prerequisites

- [Flutter SDK](https://docs.flutter.dev/get-started/install) ≥ 3.0
- A connected Android / iOS device or emulator

### Steps

```bash
# 1. Clone the repository
git clone https://github.com/your-username/ziggy.git
cd ziggy

# 2. Fetch dependencies
flutter pub get

# 3. Run on your connected device
flutter run
```

To build a release APK:

```bash
flutter build apk --release
```
